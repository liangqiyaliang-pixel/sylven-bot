# Sylven Vault — Phase 1 视觉 Mockup

## 打开方式

双击 `index.html`，在浏览器里直接预览。无需服务器，无需 npm。

## Phase 1 说明

纯静态 HTML 单文件（CSS + JS 内嵌），全部使用假数据，不连后端。

包含 5 个 Tab：

| Tab | 中文名 | 内容 |
|-----|--------|------|
| 1 | 沐栖大脑 | 14 类记忆卡片 + SVG 记忆网络图占位（Phase 4 标记） |
| 2 | 学习材料库 | 上传区 + 4 张材料卡（进度条） |
| 3 | 错题本 | 雅思/日语/设计史分科 + 4 题例 + 再考一次按钮 |
| 4 | 学习计划 | 今日任务 + 7 日周历 + 3 个长期目标 |
| 5 | 沐栖陪练 | 对话区 + 快捷按钮 + 18 天连续打卡 |

## 设计规格

```
--bg: #F8F4EC          背景（羊皮纸白）
--text: #3C2E26        主文字
--sylven: #D9744F      品牌橙
--study: #7A8C5C       学习绿
--card: #FFFCF5        卡片白
```
